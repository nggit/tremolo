# Secure Deployment
This folder contains the most simplest, expandable project structure based on
my Linux experience and may change over time.

Generated by my own mind, not an AI.

## Notes
- We don't `COPY` the project/app directory during build
- We don't start with `USER` in *Dockerfile*, we switch the user in the `entrypoint.sh` instead
- `UID=1000` means `dev` mode, higher uid e.g. >=10000 is a secure, production mode

## Build
Set the mode in [docker-compose.yml](docker-compose.yml), then:

```
docker compose build
```

## Run
```
docker compose up
```
